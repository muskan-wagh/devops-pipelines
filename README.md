# CI/CD Pipeline with GitHub Actions

This project demonstrates a basic CI/CD pipeline using GitHub Actions and Docker.

## What it does

* Triggers on every push to the `main` branch
* Builds the Docker image
* Pushes the image to Docker Hub automatically

## Tech Stack

* GitHub Actions
* Docker
* Docker Hub
* Node.js
* React.js

## Workflow

1. Push code to GitHub
2. GitHub Actions runs the workflow
3. Docker image is built
4. Image is pushed to Docker Hub

## Docker Image

```bash
docker pull muskan07wagh/react-docker:latest
````
